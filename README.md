# Tournament System Repository

This repository contains the full-stack **Tournament Registration & Leaderboard System** in the `Tournament-System/` folder.

## Stack

- Frontend: React, Vite, CSS, JavaScript
- Backend: Node.js, Express.js, MVC, service layer
- Database: MySQL-compatible TiDB Cloud
